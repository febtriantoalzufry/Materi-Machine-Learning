Pertemuan 1

Pengenalan Machine Learning

Machine Learning merupakan salah satu cabang dari Artificial Intelligence (AI) yang berfokus pada pengembangan sistem komputer agar mampu belajar dari data tanpa harus diprogram secara eksplisit. Artinya, sistem tidak diberikan aturan baku, melainkan belajar sendiri berdasarkan pola yang terdapat dalam data.

Pada pemrograman konvensional, seorang programmer harus menentukan aturan secara detail. Jika kondisi berubah, maka program harus dimodifikasi. Sebaliknya, pada Machine Learning, sistem dapat menyesuaikan diri dengan data baru tanpa perlu perubahan kode secara langsung.

Machine Learning bekerja dengan cara mengumpulkan data, mempelajari pola yang ada di dalam data tersebut, kemudian menggunakan hasil pembelajaran untuk membuat prediksi atau keputusan. Semakin banyak dan semakin baik kualitas data yang digunakan, maka hasil prediksi juga akan semakin akurat.

Perbedaan Machine Learning dan Pemrograman Konvensional

Pada pemrograman konvensional, input dan aturan diproses untuk menghasilkan output. Sedangkan pada Machine Learning, input dan output digunakan untuk membentuk aturan atau model secara otomatis.

Jenis-Jenis Machine Learning

  • Supervised Learning
  • Unsupervised Learning
  • Semi-Supervised Learning
  • Reinforcement Learning

Contoh Penerapan Machine Learning

Machine Learning banyak digunakan dalam kehidupan sehari-hari tanpa disadari, seperti pada sistem rekomendasi video di YouTube, rekomendasi produk di e-commerce, pengenalan wajah pada smartphone, hingga sistem navigasi dan kendaraan otonom.

Contoh kasus:
Pada aplikasi YouTube, sistem Machine Learning mempelajari video apa saja yang sering ditonton pengguna, durasi tontonan, serta interaksi pengguna. Berdasarkan data tersebut, sistem akan merekomendasikan video lain yang memiliki pola serupa.

Pertemuan 2

Data Preprocessing

Data preprocessing adalah proses pengolahan data mentah agar menjadi data yang bersih, terstruktur, dan siap digunakan dalam proses Machine Learning. Tahapan ini sangat penting karena kualitas data sangat berpengaruh terhadap akurasi dan performa model.

Data mentah sering kali mengandung nilai kosong, data ganda, kesalahan input, atau perbedaan skala antar fitur. Jika data tersebut langsung digunakan tanpa preprocessing, maka model Machine Learning dapat menghasilkan prediksi yang tidak akurat.

Tahapan Data Preprocessing

  • Data Cleaning – Menghapus data duplikat, noise, dan data tidak konsisten
  • Handling Missing Value – Mengisi atau menghapus data yang bernilai kosong
  • Encoding – Mengubah data kategorikal menjadi numerik
  • Normalization – Menyamakan skala data agar tidak terjadi dominasi fitur

Rumus Normalisasi Min-Max

X' = (X − Xmin) / (Xmax − Xmin)

Contoh kasus:
Dataset nilai mahasiswa memiliki kolom usia dan nilai ujian. Karena skala usia dan nilai berbeda, maka diperlukan normalisasi agar model tidak berat sebelah pada salah satu fitur.

Pertemuan 3

Supervised Learning

Supervised Learning merupakan metode Machine Learning yang menggunakan data berlabel, yaitu data yang memiliki pasangan input dan output. Model belajar dengan cara membandingkan hasil prediksi dengan nilai sebenarnya.

Jika prediksi yang dihasilkan masih salah, maka model akan menyesuaikan parameter agar kesalahan tersebut dapat diminimalkan pada proses pelatihan selanjutnya.

Jenis Supervised Learning

  • Klasifikasi – Output berupa kelas atau kategori
  • Regresi – Output berupa nilai numerik

Rumus Error (Mean Squared Error)

MSE = (1/n) × Σ(y − ŷ)²

Contoh kasus:
Menentukan apakah email termasuk spam atau tidak berdasarkan data email yang sudah diberi label spam dan non-spam.

Pertemuan 4

K-Nearest Neighbor (KNN)

K-Nearest Neighbor (KNN) adalah algoritma Supervised Learning yang menentukan kelas suatu data berdasarkan kedekatan dengan data latih di sekitarnya. KNN tidak membangun model, tetapi menggunakan seluruh data latih.

Nilai K menunjukkan jumlah tetangga terdekat yang digunakan dalam proses klasifikasi. Kelas dengan jumlah terbanyak dari K tetangga tersebut akan menjadi hasil prediksi.

Rumus Euclidean Distance

d = √((x1 − x2)² + (y1 − y2)²)

Kelebihan dan Kekurangan

  • Mudah dipahami dan diimplementasikan
  • Tidak membutuhkan training model
  • Lambat jika dataset sangat besar

Contoh kasus:
Menentukan minat olahraga seseorang berdasarkan usia dan jam olahraga dengan membandingkan data orang lain yang paling dekat.

Pertemuan 5

Naive Bayes

Naive Bayes adalah algoritma klasifikasi probabilistik yang menggunakan Teorema Bayes dengan asumsi bahwa setiap fitur bersifat independen satu sama lain.

Walaupun asumsi tersebut sederhana, Naive Bayes terbukti efektif dalam klasifikasi teks seperti email spam dan analisis sentimen.

Rumus Teorema Bayes

P(A|B) = (P(B|A) × P(A)) / P(B)

Kelebihan Naive Bayes

  • Komputasi cepat
  • Cocok untuk data teks
  • Efisien untuk dataset besar

Contoh kasus:
Jika kata “gratis” sering muncul pada email spam, maka probabilitas email tersebut diklasifikasikan sebagai spam akan meningkat.

Pertemuan 6

Perceptron

Perceptron merupakan model jaringan saraf tiruan paling sederhana yang digunakan untuk klasifikasi data linear. Algoritma ini meniru cara kerja neuron pada manusia.

Perceptron bekerja dengan mengalikan input dengan bobot, menjumlahkannya, kemudian melewati fungsi aktivasi untuk menghasilkan output.

Rumus Perceptron

y = w1x1 + w2x2 + … + b

Keterbatasan

  • Hanya mampu menangani data linear
  • Digunakan untuk klasifikasi biner

Contoh kasus:
Menentukan apakah mahasiswa lulus atau tidak berdasarkan nilai tugas dan nilai UAS.

Pertemuan 7

Linear Regression

Linear Regression adalah algoritma Machine Learning yang digunakan untuk memprediksi nilai numerik dengan memodelkan hubungan antara variabel independen dan variabel dependen dalam bentuk garis lurus.

Model berusaha mencari garis terbaik yang meminimalkan selisih antara nilai prediksi dan nilai sebenarnya.

Rumus Regresi Linear

y = mx + b

Penerapan Linear Regression

  • Prediksi harga rumah
  • Peramalan penjualan
  • Analisis tren data

Contoh kasus:
Memprediksi harga rumah berdasarkan luas bangunan, di mana semakin luas rumah maka harga cenderung semakin tinggi.
